# Quickstart

```
export SNOWFLAKE_CORTEX_TOKEN=<your-PAT-or-JWT>
export SNOWFLAKE_CORTEX_BASE_URL=https://<account>.snowflakecomputing.com

inspect eval inspect_evals/<benchmark> --model cortex-agents/<DB>/<SCHEMA>/<AGENT> --limit 1
inspect view
```
